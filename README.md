## 참여 방법

1. github에 가입해서 스터디장에게 가입한 username 혹은 email 주소를 보낸다.
2. 초대가 오면 수락한다.
3. 우측 상단의 파일 추가 기능을 이용해서 매주의 과제를 업로드한다. (git을 쓸 줄 아시면 commit&push도 환영!)
